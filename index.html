Minha tecnologia favorita

CSS

CSS é a sigla para o termo em inglês Cascading Style Sheets que, traduzido para o português, significa Folha de Estilo em Cascatas. O CSS é fácil de aprender e entender e é facilmente utilizado com as linguagens de marcação HTML ou XHTML.


Como ela ajuda as pessoas?

O CSS ajuda os desenvolvedores a estilizar pagínas HTML com diversos estilos, dando aquela incrementada no visual, você pode criar por exemplo sites mais interativos, jogos, animações e muitos efeitos legais.


Qual o grande diferencial?

Antes de usar CSS, toda a estilização tinha que ser incluída na marcação HTML. Isso significa que você deveria descrever separadamente todo o plano de fundo, as cores das fontes, os alinhamentos, etc. Mas o CSS permite que você estilize tudo em um arquivo diferente, criando assim o estilo separadamente. E, mais tarde, faça integração do arquivo CSS na parte superior da marcação HTML. Isso mantém a marcação HTML limpa e fácil de manter.


Por que eu devo usar essa tecnologia?

  • Controle do layout de vários documentos a partir de uma folha de estilos;
  • Maior precisão no controle do layout;
  • Realiza a separação entre o formato e o conteúdo de um documento;
  • Estilização dinâmica de documentos;
